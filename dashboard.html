💰 Comment recevoir votre argent via PayPal
1
Créer un compte PayPal Business
Allez sur paypal.com et créez un compte "Business" (gratuit). Vous recevrez l'argent directement sur ce compte.
2
Obtenir votre Client ID PayPal
Connectez-vous sur developer.paypal.com → "My Apps & Credentials" → "Create App" → Copier le Client ID LIVE.
developer.paypal.com/developer/applications
3
Remplacer dans kits-store.html
Dans votre fichier kits-store.html, trouvez cette ligne et remplacez sb par votre vrai Client ID :
<script src="https://www.paypal.com/sdk/js?client-id=VOTRE_CLIENT_ID¤cy=EUR&locale=fr_FR">
4
Tester en mode sandbox
Gardez sb pour les tests, remplacez par votre vrai ID uniquement quand vous mettez le site en ligne.
5
Retirer l'argent
Les paiements arrivent sur votre compte PayPal. Depuis paypal.com → "Retirer" → vers votre compte bancaire. Délai : 1 à 3 jours ouvrés. Frais PayPal : ~3.4% + 0.35€ par transaction.

💳 Stripe
Alternative à PayPal. Frais similaires (~1.5% EU). Nécessite une intégration technique plus avancée.

Affichez votre IBAN pour paiement manuel. Plus lent mais sans frais. Idéal pour les grosses commandes.
